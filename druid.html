What is Druid?

Druid is an analytics data store built to allow fast (“real-time”) access to large sets of read-oriented data.

Real Time Ingestion

Real time query. Typical analytics databases ingest data via batch processing; while the ingestion to query latency this is low for smaller data sets, Druid's real time nodes allow querying against multiple very large data sets while they are streaming into the system.

Scalable

Efficient in-memory configuration. While the entire database can be configured in memory, this becomes cost prohibitive considering the explosive growth of "Big Data." Druid query nodes are configurable and can be configured to be completely in-memory, only from storage or both.

Highly Available. Scaling up or down, replicating nodes, or recovering from failure typically impacts availability and performance. Druid uses a distributed architecture that allows replication at the segment level – relieving the load on "hot segments." And, because of replication, Druid supports rolling deployments and restarts. Scaling up or scale down just by adding or remove nodes; without re-indexing.

Human Responsive Interaction

Ad hoc, multi-dimensional filtering. Druid uses bitmap indices with CONCISE compression to create highly dense and accurate indices allowing for an arbitrary number of filters, cost effectively and without impact on performance.
http://metamarkets.com/2012/druid-bitmap-compression/

Column-oriented for speed. Data is laid out in columns so that scans are limited to specific data being searched. Compression decreases overall data footprint. http://metamarkets.com/2011/druid-part-deux-three-principles-for-fast-distributed-olap/

Seamless query against real-time and historical data.Druid automates the migration of real-time data to historical query nodes as the data ages. This allows querying against both real-time and vast amounts of historical data in a single query.

Where did Druid Come From?

Druid was created out of necessity by Metamarkets, a company focused on providing real-time interactive insight to the RTB (real time bidding) AdTech space with a full stack analytics service. It’s thesis was to create a read oriented database that could query while ingesting data, provide ad-hoc N-dimensional drill down and still provide sub-second responses. Stretching it further, the system was designed to scale without downtime, and utilize cloud resources cost efficiently.

Druid was open-sourced to help raise awareness for true real-time interactive analytics among all the "big-data" incumbents.

What's Druid Used For?

Druid is a purpose built database that provides deep exploratory insight against very large quantities of data as its being streamed or ingested into the system. It's useful for immediate insight for lots of streaming data and/or ad hoc n-dimensional filtering.

Immediate insight to large quantities of data:

The low time latency between when data is ingested into Druid and when that data is available for query allows users to develop insight very close to actual events producing that data.

Deep, exploratory drill-down:

Users value the ability to create many arbitrary filter dimensions without impacting performance, breaking scalability or cost viability (compute infrastructure). Ad hoc drill down on immediate and historical data allows users to query both "fresh", or immediately ingested data and historical data all at once.

Who is using Druid?

Metamarkets

Druid is the primary data store for Metamarkets’ full stack visual analytics service for the RTB (real time bidding) space. Pulling in over xx billion records per day, Metamarkets is able to provide insight to its customers with complex ad-hoc queries with an average query time of less than 400 milliseconds.

R-users

The R community leverages Druid for research on extremely large data sets; the ability of being able to return complex query drill downs in seconds oppose to hours helps Data Scientists be more efficient and effective. To make it even easier, Metamarkets recently open sourced the Druid-R connector (found here: https://github.com/metamx/RDruid)

Virtustream

Virtustream, an Enterprise Class Cloud provider, offers secure highly available, flexible, infrastructure as a Service (IaaS) platform, uses Druid as a real-time analytics service offering for their customers.
