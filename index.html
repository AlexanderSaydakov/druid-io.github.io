Druid

Open-source infrastructure for Real-time Exploratory Analytics on Large datasets

Download »

Druid is an open-source analytics data store designed for real-time exploratory queries on large-scale event data. Druid provides for cost-effective, highly-available real-time data ingestion and arbitrary data exploration.

Interactive

Query data as it’s being ingested for both immediate and historical insights. Ad-hoc slice-n-dice over N-dimensional data with consistent, fast response times.

Scalable

Scalable and available; consistent and predictable performance over large data sets. Scale up to increase computation power, scale down to save on costs.

Built for Analytics

Druid excels at aggregations and supports a variety of filters, calculations, and query types.
